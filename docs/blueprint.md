# **App Name**: LankanServ

## Core Features:

- Service Provider Profiles: Allow carpenters, masons, salons, and other service providers to create and manage profiles.
- Service Listing and Search: Enable users to search and browse service providers by category, location, and other relevant criteria.
- Booking and Scheduling: Allow users to book appointments or request services directly through the app.
- Review and Rating System: Enable clients to provide ratings and reviews for service providers after obtaining services.
- AI-Powered Recommendation Engine: Leverage AI to recommend the most appropriate service providers based on user requirements and preferences. The tool uses available data such as historical ratings and user location to customize its recommendations.
- Secure Messaging: Facilitate direct communication between users and service providers for inquiries and confirmations.

## Style Guidelines:

- Primary color: Deep teal (#008080), reminiscent of Sri Lanka's vibrant landscapes.
- Background color: Light, desaturated teal (#E0F8F8) to create a clean and calm user experience.
- Accent color: Warm orange (#FF7700) to highlight key interactive elements.
- Body font: 'PT Sans', a modern and accessible sans-serif for readability.
- Headline font: 'Playfair', an elegant serif font for headings and titles to convey elegance and professionalism. For body copy, use 'PT Sans'.
- Use clean, modern icons that represent different service categories.
- Simple and intuitive layout with clear calls to action.
